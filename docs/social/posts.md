# Post

🚀 I built Hiro Task Manager, an open source tool for a problem I kept facing while building software with AI agents.

Too many agent tabs. Scattered markdown plans. Messy todo files. And now, ideas getting executed at AI speed.

🧩 Hiro Task Manager is the glue that holds all of that together.

There are already plenty of task managers out there. But this one is built for human + AI collaboration.

🌐 In the web UI, I can brainstorm tasks, plan releases, prioritize work, and organize everything across boards, lists, and task groups.

🤖 In my IDE, my AI agent can find high-priority tasks, help plan the next release, execute the plan, and write detailed markdown updates back into the tasks.

🛠️ It is still in active development, and I am already using it to build itself.

🔗 Check it out on GitHub:
https://github.com/hiro-league/hirotaskmanager

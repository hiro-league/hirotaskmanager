# Task completion — voice clips

Short voice clips (e.g. praise words) live here. The app picks them **sometimes** (~18% of completions); most of the time it plays applause from `public/audio/applaud/`.

Edit `TASK_COMPLETE_VOICE_URLS` in `src/client/gamification/completionRewards.ts` if you rename files.

Applause URLs are listed in `APPLAUD_SOUND_URLS` in the same file.
